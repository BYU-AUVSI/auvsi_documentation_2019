\section*{Introduction}
This artifact depicts the wiring connections between electrical components within the airframe. 
Each connection lable corresponds to a page in the diagram and a connection on that page, so WD5.1 refers to connection 1 on page 5.
Connections that appear on more than one page are given the page number of the page that best depicts that connnection.
Page~\pageref{top} is a logical diagram showing how all of the components are connected together. 
The remaining pages are labeled photographs of the connections.
